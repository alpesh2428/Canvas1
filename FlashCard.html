ADDIE Model Flashcards

Analysis Phase
Identify goals, learner traits, constraints, delivery methods, and project timeline.
Design Phase
Blueprint learning structure, assessments, and visuals. Includes strategy and prototypes.
Development Phase
Create actual materials, test them, and revise based on feedback to enhance engagement.
Implementation Phase
Train facilitators, deploy the course, and verify functionality and readiness for use.
Evaluation Phase
Measure success using reactions and learning assessments; identify performance gaps.
